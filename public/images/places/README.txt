Folder ini digunakan untuk menampung asset gambar tempat ibadah untuk Mini-Game Sila 1 (Tebak Rumah Ibadah).

Taruh file gambar Anda di folder ini dengan nama-nama berikut sesuai dengan format gambar yang disukai (misalnya format .png atau .jpg). Gunakan format gambar dengan aspek rasio 4:3 (contoh resolusi rekomendasi: 800x600 piksel) agar pas saat dipotong menjadi grid puzzle 4x3.

Nama file yang dibutuhkan:
1. masjid.png    (untuk Masjid)
2. gereja.png    (untuk Gereja)
3. pura.png      (untuk Pura)
4. vihara.png    (untuk Vihara)
5. klenteng.png  (untuk Klenteng)

Catatan:
Sistem mini-game memiliki fitur fallback otomatis. Jika Anda belum menaruh gambar di sini atau gambar gagal dimuat, game akan otomatis menampilkan ilustrasi vektor SVG sebagai cadangan agar game tetap dapat dimainkan dengan lancar.
